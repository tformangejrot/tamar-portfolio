What is this report?

Nairobi's boutique fitness scene is growing rapidly. This report provides an overview of the Nairobi boutique fitness market, the evolution of different modalities, and their geographic distribution. Unlike the Paris report which used ClassPass data, this report is based on Google Maps/Places API searches, so while exact studio counts may vary, the relative percentages and growth trends capture real insights about the fitness landscape in Kenya's capital.
